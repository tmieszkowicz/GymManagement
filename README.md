# Gym Management API

A project exploring core concepts of Clean Architecture.

## Features

- **.NET 10** backend
- **ASP.NET Web API** as the presentation layer
- **Custom Mediator** implementation
- **CQRS** pattern
- **Custom error handling** using a **Result** pattern
- **Repository** and **Unit of Work** patterns
- **Entity Framework Core** with **SQLite** as the database
- **Strongly typed enums**
